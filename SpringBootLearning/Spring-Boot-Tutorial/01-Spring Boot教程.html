                                  # 新建Spring Boot项目

                                  Spring Boot是为了简化开发而集成的众多组件。

                                  本文将描述Spring Boot项目如何新建流程。

                                  # 是什么

                                  SpringBoot 使得创建独立的、基于 Spring 的生产级应用程序变得非常容易，你可以“运行”它。

                                  # 怎么办

                                  # 生成项目

                                  • 打开网址https://start.spring.io/open in new window

                                  选择对应的依赖，然后点击generate生成项目，下载然后解压到目录。

                                  • idea 新建
                                  • 项目设置

                                  新建完成。

                                  # 项目结构

                                  • 一般结构
                                  com
  +- example
    +- project
      +- Application.java
      |
      +- domain
      |  +- Customer.java
      |  +- CustomerRepository.java
      |
      +- service
      |  +- CustomerService.java
      |
      +- controller
      |  +- CustomerController.java
      |
      +- config
      |  +- swagerConfig.java
      |
                                  • Application 启动文件
                                  • domain 实体类
                                  • service 服务
                                  • controller 控制层
                                  • config 配置信息
